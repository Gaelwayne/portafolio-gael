# Portfolio Abraham Gael Herrera

Portafolio web personal construido con React, Tailwind CSS y Vite.

## 🚀 Características

- ✨ Diseño moderno con tema oscuro
- 📱 Completamente responsive
- ⚡ Single Page Application (SPA)
- 🎨 Animaciones suaves y efectos de hover
- 🔗 Navegación suave entre secciones
- 📊 Timeline interactiva de experiencia
- 🎯 Sección de proyectos destacados
- 💼 Información de contacto integrada
- 🌐 Sistema de internacionalización (Español/Inglés)
- 🎨 Iconos SVG personalizados para tecnologías
- 📱 Botón de WhatsApp directo

## 🛠️ Tecnologías

- **React** - Librería de UI
- **Vite** - Build tool
- **Tailwind CSS** - Framework de CSS
- **Lucide React** - Iconos

## 📦 Instalación

1. Instala las dependencias:
```bash
npm install
```

2. Inicia el servidor de desarrollo:
```bash
npm run dev
```

3. Abre tu navegador en `http://localhost:5173`

## 🏗️ Build para producción

```bash
npm run build
```

Los archivos de producción se generarán en la carpeta `dist/`.

## 📋 Estructura del proyecto

```
portfolio-gael/
├── src/
│   ├── components/
│   │   ├── Navbar.jsx       # Barra de navegación fija
│   │   ├── Hero.jsx          # Sección de presentación

│   ├── main.jsx              # Punto de entrada

└── vite.config.js
```

## 🎨 Personalización

Para personalizar el portafolio:

1. **Colores**: Edita `tailwind.config.js` para cambiar la paleta de colores
2. **Contenido**: Actualiza los arrays de datos en cada componente
3. **Estilos**: Modifica `src/index.css` para ajustes globales
4. **Enlaces sociales**: Actualiza las URLs en `Hero.jsx`

## 📱 Secciones

- **Inicio**: Presentación con nombre, título y enlaces sociales
- **Experiencia**: Timeline con historial laboral completo
- **Proyectos**: Grid de proyectos destacados con tecnologías
- **Sobre mí**: Habilidades técnicas organizadas por categorías
